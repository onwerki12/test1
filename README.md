# Campus Heart MVP

Godot 4 기반의 초소형 육성 미연시 프로토타입입니다.

## 포함된 기능
- 7일 일정 진행
- 행동 선택: Study / Workout / Talk / Rest
- 스탯: 지식, 체력, 매력, 스트레스, 호감도
- 중간 이벤트 2개
- 엔딩 분기 4종

## 실행 방법
1. Godot 4.x에서 이 폴더를 엽니다.
2. `project.godot`를 선택합니다.
3. 실행하면 됩니다.
