# Campus Heart Web MVP

아이폰 환경에서도 설계하고 테스트할 수 있게 바꾼 **웹 기반 육성 미연시 프로토타입**입니다.

## 지금 들어간 기능
- 7일 일정 진행
- 행동 선택: 공부 / 운동 / 대화 / 휴식
- 스탯: 지식, 체력, 매력, 스트레스, 호감도
- 중간 이벤트 3개
- 엔딩 분기 4종
- 브라우저 `localStorage` 저장 / 불러오기
- 모바일 친화 UI

## 파일 구조
- `index.html` : 화면 뼈대
- `style.css` : 모바일 우선 UI 스타일
- `game.js` : 게임 상태, 이벤트, 엔딩, 저장/불러오기 로직

## iOS에서 보는 법
### 가장 쉬운 방법
1. GitHub에서 `index.html` 파일을 연다.
2. raw 보기 또는 Pages 배포 후 Safari에서 접속한다.

### 추천 방법: GitHub Pages
1. 레포의 **Settings**로 간다.
2. **Pages** 메뉴를 연다.
3. Source를 `Deploy from a branch`로 선택한다.
4. Branch를 `main` / root로 설정한다.
5. 저장 후 몇 분 기다린다.
6. 배포된 URL을 Safari로 열면 바로 플레이 가능하다.

## iOS만으로 개발하는 추천 루프
1. 나한테 기능 추가 요청
2. 내가 레포 파일 수정
3. 너는 Safari에서 페이지 새로고침해서 확인
4. 이상한 점 있으면 바로 수정 반복
